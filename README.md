## What is this?
Simple Sinatra application for a mounted touch screen in my NYC apartment.

### Technologies Used
- Ruby
- Sinatra
- [Wunderground wrapper gem](https://github.com/wnadeau/wunderground) to hit [wunderground.com's](http://wunderground.com) API
- [Nokogiri](http://www.nokogiri.org/) to parse the [MTA's XML file](http://web.mta.info/status/serviceStatus.txt)
- [PEXEL's API](https://www.pexels.com/api/) for random background pictures
